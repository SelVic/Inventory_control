# Inventory_control

Web-приложение для складского учета


### Установка
```
git clone https://github.com/SelVic/Inventory_control
В корневой папке проекта
npm install
node server

Приложение развернется на localhost:3000

Проект использует webpack и gulp
Если хотите изменить код, то для того, чтобы внесенные изменения в вступили в силу, то — 
В консоли, в корневой папке:

-Для изменений JavaScript: webpack --watch
-Для изменений CSS/SCSS: gulp watch
```


### Функционал:
```
-Добавление новых наименований в базу данных на MondoDB
-Редактирование количества существующих предметов с помощью списаний / добавлений
-Валидация введенных данных
-Вывод всех наименований в таблицу для контроля и наглядности
-Вывод всех историй о списании/добавлении у конкретного предмета в таблицу 
(Кнопка стрелки у соответствующего предмета в таблице)
-Удаление предмета из базы данных
(Кнопка урны у соответствующего предмета в таблице)
-Удаление истории о предмете
(Кнопка "Удалить" у соответствующей операции)
-Фильтр по наименованиям
```
